# Hybrid AI Image & Carousel Engine

This project is an automated design-to-code pipeline for generating high-fidelity LinkedIn carousels and social media infographics. It combines AI-generated background textures with pixel-perfect CSS typographic overlays.

## 🚀 The Antigravity Journey
Built entirely within the Antigravity agentic environment, this project demonstrates a shift from simple "AI image generation" to a **Hybrid Workflow**:
1. **AI Synthesis**: Using DALL-E/Imagen to create clean, minimal background textures or editorial photography.
2. **CSS Layouts**: Wrapping those assets in a rigorous HTML/CSS design system to ensure brand-aligned typography and spacing.
3. **Automated Export**: A Puppeteer-based script that renders the HTML "canvas" into professional PNG files ready for posting.

## 🛠 Tech Stack
- **Core**: HTML5, CSS3 (Vanilla)
- **Engine**: Puppeteer (Node.js) for high-resolution rendering
- **Logic**: JavaScript
- **Framework**: Antigravity Multi-Agent Design Protocol

## 📁 Key Components
- `carousel.html`: The multi-slide source file containing the design and typography.
- `export_carousel.js`: The automation script that takes screenshots of each slide at 1080x1350 resolution.
- `.agents/skills/design/`: A structured knowledge base governing the **Visual Identity** (typography, colors, layouts) and the **Technical Workflow**.
- `Image references/`: A collection of curated brand archetypes used to ground the AI's creative output.

## 📈 Results
The output of this system is professional-grade imagery that maintains strict brand consistency across all slides—something pure AI generation often fails to achieve.

### Example Outputs:
- `slide_1.png` - `slide_5.png`: A complete 5-slide carousel for LinkedIn.
- `syd-life-infographic.html`: A data visualization layout for "Syd Life" branding.
- `syd-viz-X.png`: Rendered isotype grids and journey lines.
