# Straunt – Visual Identity Guide

This document captures Straunt's complete visual identity as observed on their website (straunt.ai). Use this as the reference for any design, presentation, or visual content created on behalf of Straunt.

---

## Logo

- **Wordmark**: "STRAUNT" in bold, uppercase, dark text using the primary font.
- **Icon**: A stylized **square/diamond shape** in the brand orange (`#F97A34`), used as a standalone mark beside the wordmark.
- **Placement**: Top-left of the navigation bar, always combined (icon + wordmark).
- **Background**: The nav bar is white, making the orange logo highly visible.

---

## Color Palette

### Primary Brand Color
| Name | Hex | Usage |
|------|-----|-------|
| **Brand Orange** | `#F97A34` | Primary CTA buttons, logo icon, accent text highlights in headlines, key section backgrounds |
| **Hover Orange** | `#E06F20` | Button hover state — slightly darker shade of brand orange |
| **Pale Orange Tint** | `#FFF6ED` | Subtle background tint used behind orange icons or callout elements |

### Background Colors
| Name | Hex | Usage |
|------|-----|-------|
| **White** | `#FFFFFF` | Primary page background, card backgrounds, nav bar |
| **Light Gray** | `#F5F7FB` | Alternating section backgrounds; creates visual separation without a harsh contrast |

### Text Colors
| Name | Hex | Usage |
|------|-----|-------|
| **Heading Black** | `#161616` | All major headings (H1, H2) — near-black for maximum contrast |
| **Body Dark Gray** | `#1F2937` | Body paragraphs, feature descriptions, card text |

### Color Usage Summary
- The site is **light-mode first** — white and light gray dominate.
- Orange is used **sparingly** but powerfully as the single accent color. It creates immediate visual hierarchy.
- There is **no use of blue, green, or purple** in the primary palette.
- Dark sections (if any) use very dark near-black backgrounds, not true black.

---

## Typography

### Primary Font Family
> **Lexend** (sans-serif)

- A modern, geometric sans-serif font designed specifically to improve reading speed and comprehension.
- Available from Google Fonts.
- Used across **all text elements** — headings, body, buttons, nav, captions.

### Type Scale (Approximate from Website)
| Element | Style |
|---------|-------|
| **H1 Hero** | ~48–56px, Extra Bold (800), dark near-black |
| **H2 Section Heading** | ~36–42px, Bold (700), dark near-black |
| **H3 Sub-heading** | ~22–28px, Semi-Bold (600) |
| **Body / Descriptions** | ~16–18px, Regular (400), dark gray |
| **Feature Bullet Titles** | ~14–16px, Semi-Bold (600) |
| **Button Text** | ~15–16px, Bold (700), white |
| **Caption / Label** | ~12–13px, Regular or Medium |

### Key Typography Behaviors
- **Accent color in headlines**: Key words or phrases within H1/H2 headlines are set in brand orange (`#F97A34`) to create emphasis. Example: *"The full-stack **Restaurant OS**"* — "Restaurant OS" renders in orange.
- **Letter spacing**: Tight in headlines, creating a premium, confident feel.
- **Line height**: Relaxed in body text for readability.
- **All caps**: Used sparingly — only in testimonial attributions (e.g., *"— HANOI SPICE HOUSE"*) for stylistic contrast.

---

## Button Design

### Primary CTA Button
- **Shape**: Pill / fully rounded (`border-radius: 9999px`)
- **Background**: Brand orange (`#F97A34`)
- **Text**: White, Bold, ~15px
- **Padding**: Generous horizontal padding (~24–32px) creating a wide, prominent button
- **Hover**: Slightly darker orange (`#E06F20`)
- **Example**: "Book A Demo" button in the top-right nav

### Secondary Button
- **Shape**: Same pill shape
- **Style**: Ghost/outline or solid in a muted tone
- **Example**: "Deny" button in cookie banner — muted gray background

### Inline CTAs / Text Links
- Use arrow `→` symbol alongside text links (e.g., "Read More →")
- No underlines in nav — nav links rely on spacing and hover color change

---

## Layout & Spacing

- **Layout type**: Wide, centered content blocks with generous lateral white space.
- **Max content width**: ~1200px centered container.
- **Section rhythm**: Alternates between white (`#FFFFFF`) and light gray (`#F5F7FB`) sections for visual separation without hard borders.
- **Hierarchy**: Clear H1 → Sub-copy → CTA structure in every hero section. No clutter.
- **Card-based UI**: Feature cards, business type cards, and solution modules all use **rounded rectangle cards** with soft shadows.
- **Breathing room**: Significant padding between sections — the design never feels cramped.

---

## Iconography & Motifs

### Hexagonal Shape System
- A **recurring hexagonal container** is used to display partner/integration logos and some feature icons.
- This is a distinctive visual motif unique to Straunt — sets them apart from competitors using plain circles or squares.
- Hexagons are rendered in muted neutral tones (gray, light orange) with the logos/icons inside.

### Icon Style
- **Line-art style icons** — clean, modern, minimal stroke.
- Color: Primarily brand orange or dark gray.
- Used for feature bullet points and solution module identifiers.
- No illustrative/cartoon-style icons.

---

## Imagery Style

### Photography
- **High-resolution food photography**: Styled shots of actual restaurant food (burgers, bowls, sushi, etc.)
- **Warm, saturated tones**: Photos skew warm — golden hour lighting that makes food look appetizing.
- **Context shots**: Some photography shows the inside of restaurants in operation (busy kitchens, tables).
- **Purpose**: Grounds the AI/tech product in the real physical world of the restaurant. Makes it feel human and familiar.

### UI Mockups / Product Screenshots
- Dashboard screenshots and product UI are shown in context (on screens, in mockup frames).
- Product UI appears to use a clean, dark-background dashboard with orange accents (consistent with brand).

---

## Navigation Design

- **Style**: Minimal, flat top nav bar.
- **Background**: White.
- **Link style**: Dark text, no underlines, hover changes color.
- **Dropdown menus**: Appear on hover for "Business Types," "Solutions," "Agents," and "Resources."
- **CTA in nav**: "Book A Demo" — pill-shaped orange button, always visible top-right.
- **Nav items**: Business Types | Solutions | Agents | Features | Pricing | Resources

---

## Testimonial Block Design

- Testimonial quotes are displayed in **orange italic text** — visually distinct and emotionally warm.
- Attribution is in **small all-caps** text: *"— HANOI SPICE HOUSE"*
- Presented on a muted background (light gray section) to contrast with the white main content.

---

## Metrics / Stats Display

- Large numbers (700+, 100+, 15–30%, etc.) are displayed in **very large, bold orange type** as headlines.
- Supporting explanation text sits directly below in standard size.
- This format signals credibility and draws the eye immediately to proof points.

---

## Overall Design Character

| Attribute | Straunt's Style |
|-----------|----------------|
| **Mode** | Light (not dark) |
| **Vibe** | Clean, modern, confident — not flashy |
| **Complexity** | Low — minimal visual noise |
| **Decoration** | Minimal — function over decoration |
| **Color count** | Very restrained — orange + white + gray only |
| **Animations** | Subtle — no aggressive motion |
| **Feel** | Premium SaaS, but restaurant-native |
